# ForgeFit 🧠💪

ForgeFit is an AI-powered personal fitness and wellness tracking platform built using the MERN stack and modern AI integrations.

The platform helps users track workouts, diet, hydration, recovery, wellness scores, and daily habits while generating personalized AI-driven fitness insights and plans.

---

## 🚀 Features

### 🏋️ Fitness Tracking
- Exercise logging
- Workout goal management
- Daily wellness tasks
- Progress monitoring

### 🥗 AI Diet Planner
- AI-generated diet plans
- Personalized nutrition suggestions
- Wellness-based meal recommendations

### 🤖 AI Features
- AI-generated fitness suggestions
- Daily wellness score
- Smart recovery analysis
- Adaptive health recommendations

### 📊 Future Planned Features
- AI health timeline
- Burnout and recovery detection
- Animated AI health avatar
- Mood and energy tracking
- Smart analytics dashboard
